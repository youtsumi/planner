API syntax

mode={event|candidate|submit|admin}

?mode=event

Get event list. eventid must be specified.
?mode=event&eventid=G268556

?mode=candidate

Get candidate list. eventid must be specified.
?mode=candidate&eventid=G268556

?mode=submit

Submit "state" string for specified galaxy.
?mode=submit&eventid=G268556&obsid=Kanata-HONIR&galid=GL080746+224854&state=Reserved;filter=i;depth=20.5

?mode=admin

Append ignore flag for galaxies who belong to eventid.
