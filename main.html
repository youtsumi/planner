Event list | Candidates | Observing Log | Admin
Need machine readable format like JSON output? Just replace "main.html" with "processor.py" or click here.

API syntax

mode={event|candidate|submit|admin}

?mode=event

Get event list. eventid must be specified.
processor.py?mode=event&eventid=G268556

?mode=candidate

Get candidate list. eventid must be specified.
processor.py?mode=candidate&eventid=G268556

?mode=submit

Submit "state" string for specified galaxy.
processor.py?mode=submit&eventid=G268556&obsid=Kanata-HONIR&galid=GL080746+224854&state=Reserved&filter=i&depth=20.5&obsdatetime=2017-02-21 12:46:40&observer=Y. Utsumi&hastransient=no

?mode=admin

Append ignore flag for galaxies who belong to eventid.
